# 💍 Convite de Casamento Digital - Ronaldo & Elaine

Bem-vindo ao repositório do nosso convite de casamento! Este projeto foi desenvolvido para compartilhar de forma elegante e interativa os detalhes do nosso grande dia.

## 🚀 Sobre o Projeto
Este é um convite digital em formato de "livro" (flipbook), onde os convidados podem folhear as páginas para descobrir nossa história, ver fotos, conferir a contagem regressiva e saber como chegar ao local da cerimônia.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura do convite.
- **CSS3**: Estilização responsiva e design elegante.
- **JavaScript**: Lógica do contador regressivo, música de fundo e efeitos de animação.
- **StPageFlip**: Biblioteca para o efeito de virar as páginas (flipbook).

## 📂 Estrutura do Projeto
- `/css`: Arquivos de estilo (incluindo responsividade).
- `/js`: Scripts de lógica e contagem regressiva.
- `/img`: Fotos do casal, capa e elementos visuais.
- `/audio`: Trilha sonora do convite.
- `/libs`: Bibliotecas de terceiros.

## 📋 Funcionalidades
- [x] Efeito de livro (Flipbook).
- [x] Música ambiente com controle de volume/pausa.
- [x] Galeria de fotos.
- [x] Contagem regressiva precisa para o dia 03/10/2026.
- [x] Mapa com rota integrada para o local do evento.
- [x] Design responsivo para celulares e desktops.
